# Luber Consumer App

Customer-facing Next.js experience that powers booking, account management, and payment flows for Luber.

## Getting Started

```bash
pnpm install
pnpm dev
```

Visit `http://localhost:3000` to verify changes. Use `.env.local` for Supabase, Stripe, and Vercel secrets before hitting any remote resources.

## Contributor Guide

Follow the shared standards documented in the root `AGENTS.md`. That file covers project structure, coding conventions, testing expectations, and PR requirements for both the B2B and consumer surfaces.
